Manuale utente

Avvio attraverso Jar eseguibile


E' possibile avviare sia il Server che il Client utilizzando i relativi file.jar e utilizzando gli argomenti previsti:

java -jar knnServer.jar
java -jar knnClient.jar


Server


Avviata l'applicazione, il server conferma la sua attività visualizzando una stringa a video, e in generale fornisce sempre un feedback su quello che sta avvenendo o è avvenuto.

Esempi:


inizio server

inizio: ServerSocket[addr=0.0.0.0/0.0.0.0,localport=2050]

Connessione avvenuta con successo
closing...


Client


Avviato, il client permette all'utente di scegliere se predirre il valore di Knn a partire da una tabella di un file, file binario o da database.

Una volta inserito il nome, il sistema effettuerà le operazioni necessarie a iniziare la fase di previsione. In questa fase l'utente deve inserire i valori continui, discreti e il valore k.
